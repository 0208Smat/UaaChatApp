
# Chat grupal UAA

Proyecto final de intro a programación móvil, app de chat en sala grupal con login.




## Instalación

Instalar ProyectoFinal-AppChat con npm, luego de clonar el repositorio

```bash
  cd AppChat
  npm install
```
    
## Despliegue

Ejecutar con ionic

```bash
  ionic serve
```
